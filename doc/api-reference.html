     Modules

    Jackpot

    the-game

     The Game

    Jackpot is a simple game of chance played by one person.

    Jackpot.AgentFirst

    A trivial strategy that always chooses the first cell in the list of choices.

    Jackpot.AgentHiLo

    HiLo implementation using an explicit priority sequence, but is quadratic in the worst case.

    Jackpot.AgentHiLo2

    HiLo implementation uses a maximum threshold then minimum. Initial chance of performing both scans on starting board is about 50%, but this goes up to 100% as 7, 8 and 9 are flipped.

    Jackpot.AgentLast

    A trivial strategy that always chooses the last cell in the list of choices.

    Jackpot.AgentMax

    A trivial strategy that always chooses the highest value cell.

    Jackpot.AgentMin

    A trivial strategy that always chooses the lowest value cell.

    Jackpot.Board

    The Jackpot board of 9 cells that can be flipped.

    Jackpot.Dice

    Simulate random die and roll of 2 dice.

    Jackpot.Game

    Play a game of Jackpot with a player agent. Run Monte Carlo simulation of many games to generate win/loss statistics.

    Jackpot.Graph

    Analyze the play of a player agent across all possible games.

    Jackpot.T

    Types for the Jackpot application.

    Utilities for the Jackpot application.
